# 1.アプリの概要
- 曲を、よくあるアーティスト単位でなく感情単位のフォルダに振り分け、
その時の感情に応じた音楽がすぐに聴けるサービスが欲しいと思い作成。
併せて、自身の気分を可視化・記録できる機能も盛り込んだ。
# 2.アプリの機能一覧
- 認証機能
- プレイリスト作成機能
- 楽曲投稿機能
- 画像投稿機能
- 感情認識機能
- グラフ作成機能
